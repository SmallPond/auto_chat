# Auto chat

程序功能：基于itchat的微信定时发送消息


# how to start
## 第三方库依赖
>1.itchat  
2.apscheduler

都可通过pip 安装
```
pip install itchat 
pip install apscheduler
```
## 使用方法：

1.在文件task_file.txt中进行配置

对配置文件task_file.txt进行配置时，可使用`#`开头注释一行文字

![](http://i.imgur.com/cu9WGIR.png)

2.在命令行运行chat.py程序
```	
python chat.py 
```
